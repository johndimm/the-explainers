The Explainers - User Guide

Welcome to The Explainers! This app helps you understand difficult texts by providing AI-powered explanations of selected passages. Whether you're tackling Shakespeare, philosophy, or classic literature, we make challenging texts accessible.

Getting Started

Quick Start

  1. Open the application and click "Library" to choose a book to read
  2. Simply select any text passage you find confusing
  3. Get an instant AI explanation tailored to your needs
  4. Ask follow-up questions in the chat interface

First-Time Setup

For the best experience, configure your profile and preferences:

Profile Setup

Click the green "Profile" button to set:

  • Your age (for age-appropriate vocabulary)
  • Preferred language (12 languages supported)
  • Education level (elementary to graduate school)

Settings Configuration

Click the blue "Settings" button to customize:

  • AI model provider (GPT-4, Claude, DeepSeek, Gemini)
  • Response length (brief, medium, long)
  • Text and chat fonts
  • Explanation style (50+ personalities!)

Core Features

Text Reader

  • Clean Interface: Distraction-free reading environment
  • Text Selection: Highlight passages with mouse or touch
  • Auto-Bookmarks: Your reading position is automatically saved
  • Search Function: Find specific words or phrases

AI Explanations

When you select text, get comprehensive explanations including:

  • Vocabulary Help: Definitions and contextual meanings
  • Structure Analysis: Breaking down complex sentences
  • Plot Context: What's happening in the story
  • Character Insights: Who's speaking and their motivations
  • Historical Background: Relevant historical context
  • Style Variations: Hover over profile photos to see larger images
  • Conversation Flow: Ask follow-up questions and build on previous explanations

How to Use

Reading a Text

  1. Select a Book: Click "Library" to browse available texts
  2. Navigate: Scroll through the text reader
  3. Select Passages: Highlight any confusing text
  4. Get Explanations: Chat interface appears automatically
  5. Ask Follow-ups: Continue the conversation

Customizing Explanations

  1. Change AI Provider: Select GPT-4, Claude, DeepSeek, or Gemini
  2. Adjust Length: Brief (2-3 sentences), medium (1-2 paragraphs), or long (detailed)
  3. Pick a Style: Choose from dozens of explanation personalities
  4. Auto-Save: All preferences automatically save when changed

Technical Information

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • Internet connection for AI explanations

Device Optimization

  • Desktop: Full-screen text reader with dedicated chat page
  • Mobile: Touch-friendly with enhanced text selection and dedicated chat page
  • Tablet: Responsive design adapts to screen size

Chat Interface

  • Dedicated Chat Page: Full conversation interface separate from text reader
  • Conversation History: All messages accumulate during your session
  • Style Indicators: See which AI model and style generated each response
  • Re-explain Feature: Get new explanations in different styles instantly

Troubleshooting

Common Issues

Text Won't Load: Check internet connection or try a different text

Selection Not Working: Ensure you're selecting complete words or phrases

AI Not Responding: Verify internet connection and try again

Wrong Language: Check Profile settings for language preferences

Getting Help

  • Use the chat interface to ask questions about the app
  • Try different AI providers if one isn't working
  • Adjust response length if explanations are too long or short
